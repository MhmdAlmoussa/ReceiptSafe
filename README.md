## Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language:** Kotlin
- **Package Manager:** Gradle

### Installation

Build ReceiptSafe from the source and install dependencies:

1. **Clone the repository:**

    ```sh
    ❯ git clone https://github.com/MhmdAlmoussa/ReceiptSafe
    ```

2. **Navigate to the project directory:**

    ```sh
    ❯ cd ReceiptSafe
    ```

3. **Install the dependencies:**

**Using [gradle](https://kotlinlang.org/):**

```sh
❯ gradle build
```

### Usage

Run the project with:

**Using [gradle](https://kotlinlang.org/):**

```sh
gradle run
```

### Testing

Receiptsafe uses the {__test_framework__} test framework. Run the test suite with:

**Using [gradle](https://kotlinlang.org/):**

```sh
gradle test
```
